# Generative_AI

A portfolio of Generative AI projects focusing on Natural Language Processing (NLP). The repository includes implementations of pre-trained word vectors, PCA on word embeddings, custom Word2Vec, summary writing, sentiment analysis, and text summarization using Python.

This repository contains hands-on implementations of fundamental Generative AI and Natural Language Processing (NLP)concepts using Python and Jupyter Notebook. The projects demonstrate practical applications of word embeddings, semantic similarity, dimensionality reduction, and information retrieval techniques.

📂 Projects Included

📌 1. Pre-trained Word Vectors
- Load and utilize pre-trained word embeddings.
- Find semantically similar words.
- Explore word relationships using vector representations.

📌 2. PCA on Word Embeddings & Word2Vec
- Train and work with Word2Vec models.
- Reduce high-dimensional word embeddings using Principal Component Analysis (PCA).
- Visualize semantic relationships between words.

📌 3. Wikipedia Information Retrieval
- Retrieve information from Wikipedia using Python.
- Fetch article summaries for user-specified topics.
- Demonstrate basic information extraction using the Wikipedia API.
